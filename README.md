# BRIGHT-COFFEE-SALES-ANALYSIS
END TO END DATA ANALYTICS PROJECT

																				📌 Project Overview

This project analyses transactional sales data from a multi-store coffee shop business to uncover trends, measure business performance, and generate actionable insights. The objective was to transform raw sales data into meaningful business intelligence using SQL for data preparation and Microsoft Excel for data analysis and visualisation.

*The analysis focuses on answering key business questions such as:
*Which stores generate the most revenue?
*Which product categories and products perform best?
*What are the peak trading periods?
*How do sales vary by month and day of the week?
*What recommendations can improve business performance?

-----------------------------------------------------------------------------------------------------------------------------------------

																				🎯 Project Objectives

The main objectives of this project were to:

1) Clean and prepare raw sales data.
2) Create calculated fields to support business analysis.
3) Analyse sales performance across stores, products, and time periods.
4) Develop an interactive dashboard using Excel.
5)Present business insights and recommendations to stakeholders.

-----------------------------------------------------------------------------------------------------------------------------------------

																				🗂 Project Planning

The project followed a structured data analytics workflow:

1. Business Understanding
   
*Identified business questions.
*Defined key performance indicators (KPIs).
*Planned the analytical approach.


3. Data Preparation (SQL)

The dataset was cleaned and transformed using SQL in Databricks.

Tasks included:

*Checking for missing values.
*Identifying duplicate records.
*Standardising text fields.
*Creating calculated columns including:
*total_revenue
*month_name
*day_name
*time_bucket

3. Data Analysis (Excel)
   
The cleaned dataset was exported to Microsoft Excel where PivotTables, PivotCharts, KPIs, and dashboards were created.

5. Data Visualisation

Interactive charts and dashboards were designed to communicate findings clearly to business stakeholders.

-----------------------------------------------------------------------------------------------------------------------------------------
																						💡 Key Findings

The analysis revealed several important business insights:

Revenue remained relatively consistent across the six-month period.

Sales were evenly distributed throughout the week, indicating stable customer demand.

Some store locations consistently outperformed others in revenue generation.

A small number of products (Top 10) contributed a significant percentage of total revenue.

Certain product categories generated substantially higher revenue than others.

Customer purchasing patterns varied throughout the day, highlighting peak trading periods that can support staffing and inventory planning.



-----------------------------------------------------------------------------------------------------------------------------------------

                                                    📌 Business Recommendations

Based on the analysis, the following recommendations were made:

Maintain sufficient inventory levels for top-performing products.

Increase staffing during peak trading hours.

Implement targeted promotions during slower trading periods.

Use successful stores as benchmarks for improving lower-performing locations.
